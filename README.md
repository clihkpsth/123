# TestApp

Простое iOS приложение с одной кнопкой "Тест".

## Описание

Приложение содержит одну кнопку, при нажатии на которую появляется алерт с сообщением.

## Сборка

### Локально

Откройте `TestApp/TestApp.xcodeproj` в Xcode и нажмите Run.

### GitHub Actions

Проект настроен для автоматической сборки через GitHub Actions. При каждом пуше в main/master ветку запускается сборка.

Собранное приложение можно скачать из артефактов workflow.
